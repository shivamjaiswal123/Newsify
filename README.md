## Technology used: 
* Kotlin
* XML
* Rest API
* MVVM Architecture
* Dependency Injection with Hilt
* ViewPager2
* Third party libraries like [retrofit](https://square.github.io/retrofit/), [glide](https://github.com/bumptech/glide), [spinkit](https://github.com/ybq/Android-SpinKit).
